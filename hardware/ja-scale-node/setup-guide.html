Software setup & calibration

3 Register the module in the portal

  1. Open Workshop → Inventory → Live Bins → ⚙ Setup → + Add module.
  2. Name it after its location (e.g. “Bin rack A — shelf 2”).
  3. Copy the device key it shows (also available later via Copy key on the module card).

4 Flash the ESP32

  1. Open hardware/ja-scale-node/ja-scale-node.ino in Arduino IDE.
  2. Library Manager → install “HX711” by Bogdan Necula (bogde).
  3. Fill in the three constants at the top: WIFI_SSID, WIFI_PASS, DEVICE_KEY.
  4. Board: Arduino Nano ESP32 (install “Arduino ESP32 Boards” core + accept driver prompts). If upload fails with a DFU/LIBUSB error: double-tap RST (LED fades), reselect the port, upload again.
  5. Open Serial Monitor (115200) — you should see [post] HTTP 200 lines; the portal card turns green within ~30 s.

5 Calibrate (all done from the portal — never reflash)

  1. Tare — with the bin platform empty (or the empty bin sitting on it, if it always stays), press Tare on the cell tile.
  2. Calibrate — put a known weight on (bathroom-scale-verified item, e.g. “2,040 g”), press Calibrate, type the grams.
  3. Unit weight — enter the part’s weight in g / unit (weigh 10 parts, divide by 10 for accuracy).
  4. Assign the bin # and part number, and set Alert ≤ for the low-stock red flag.
  5. Check it: the big number should match a hand count. Add/remove a part and watch it move.
Units = (live weight − tare) ÷ unit weight, rounded. Light parts (<50 g) count more reliably on smaller-capacity cells — a 5 kg cell resolves single washers far better than a 50 kg one.

6 Troubleshooting

SymptomCheck
Card stays red / “never seen”Serial Monitor: WiFi connect lines → SSID/password; HTTP 401 → wrong device key; no post lines at all → power/USB.
[cell 0] not ready in serialDT/SCK swapped or loose — white/green load-cell wires on A−/A+, DT→16, SCK→17.
Count drifts ±1 unitRaise AVG_SAMPLES in the sketch (10 → 25), check the platform isn’t touching anything, keep the cell out of direct fan draft.
“needs calibration” on the tileDo step 5 in order — Tare first, then Calibrate with the known weight still on.
Tare/Calibrate refuses (“stale”)Module hasn’t reported in >5 min — check it’s online, wait one heartbeat (~30 s), retry.
Weight fine, units wrongUnit weight g/unit is off — weigh 10 parts and divide.
Adding more bins: every extra bin is just another module (or a second cell on a spare module) — register, flash, tare, calibrate, name. Config always lives in the portal.
